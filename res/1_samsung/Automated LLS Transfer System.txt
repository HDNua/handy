ALT: Automated LLS Transfer
기존의 LLS 전달은 LLS 담당자가 LLS 수정 후 zip 파일로 압축하여 SharePoint에 올리고 나면, RTL 담당자가 이를 SmartFTP를 통해 VWP 내로 가져와 압축을 풀고 compile 후 벡터를 생성하는 것이었습니다. 이 부분은 사실 전체 자동화 가능한 것으로 보이며, 따라서 이 작업을 통해 자동 LLS 전송 시스템을 구축하였습니다.
* LLS : RTL 검증을 위해 사용되는 벡터를 생성하는 프로그램
* VWP : RTL 검증을 진행하는 Linux base server
* SharePoint -> Local PC, Local PC -> Windows 전송의 2 step 으로 분리하여 구현; 외부 저장소 특성을 가리지 않는 유연한 시스템 제공 (ex) git repo
* LLS를 전송하는 즉시 LLS compile을 시작하는 코드 구현
* LLS로 벡터를 생성하기 위해 필요한 입력 파일인 TC 파일도 ALT로 전송
* Pass rate 관리를 위한 Progress sheet Excel도 ALT로 전송
* 3G와 같은 legacy area에 대해서도 자동 전송 가능한 일반화된 솔루션 제공
* 다중 채널 동기화를 통해 LLS, TC, Excel을 병렬로 내려받아 관리
* 검증 담당자 부재중에도 파일 전송이 가능하여 업무 효율성 증대
* C#, Windows API, Selenium 등 다양하고 확장성 높은 도구 기반 구현
